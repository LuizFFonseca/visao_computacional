# Projeto Final - Visão Computacional

Este repositório contém os códigos e análises desenvolvidos para o projeto final da disciplina de Visão Computacional.

## Estrutura do Projeto

Foram utilizados dois scripts principais para o desenvolvimento do trabalho:

* **`labeler.py`**: Script utilizado para gerar as máscaras necessárias para a etapa de visão clássica.
* **`Relatoriov2.ipynb`**: Notebook contendo todas as análises, experimentos e resultados do trabalho.

## Ferramentas e Créditos

* **Documentação**: O **Gemini Pro** foi utilizado para auxiliar na geração de comentários e *docstrings* do código, visando manter a clareza e padronização.
